# JsCRM — 1C Platform Compatible Runtime (Bun/TS)

Server-side runtime compatible with 1C:Enterprise. Executes 1C language via **IR v1** — a JSON-based Intermediate Representation generated by a 1C exporter.

## Architecture

```
1C Configuration
    ↓ (1C Exporter)
IR v1 JSON  ←→  ir-validator (ajv)
    ↓
  VM (Bun/TS)
    ↓
 SQLite  ←→  WebSocket  ←→  Browser
```

Two execution paths:

| Path | Source | Use |
|------|--------|-----|
| Production | `export/ir/*.ir.json` (from 1C exporter) | Real modules |
| Testing | `fixtures/*.ir.json` (handwritten) | Dev & compat |

## Quick Start

```bash
bun install
bun test              # 26 tests, 0 fail
bun run src/server.ts # http://localhost:3000
```
